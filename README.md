# ML From Scratch

A from-scratch classical machine learning library focused on clear implementations, reusable architecture, and progressive CPU performance optimization.

## Goal

The goal of this project is to rebuild classical machine learning algorithms from first principles while learning:

- machine learning fundamentals
- numerical computing
- software architecture
- testing
- benchmarking
- C-based performance optimization

## Current Phase

Phase 0: project foundation.

At this stage, the project contains the repository structure, documentation, and initial Python package layout.
